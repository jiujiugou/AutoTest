# AutoTest — 自动化监控平台

## 一句话

对 HTTP/TCP/DB/Python/TEMPLATE 目标定时执行 + 断言，失败时自动触发 AI 根因分析。

## 技术栈

| 层 | 技术 |
|----|------|
| 运行时 | .NET 10 |
| 数据库 | SQL Server（主）/ SQLite（开发），Dapper |
| 缓存 & 锁 | Redis（StackExchange.Redis） |
| 调度 | Hangfire |
| 实时推送 | SignalR |
| 日志 | Serilog → Elasticsearch + Console |
| AI | OpenAI 兼容 API（豆包/Ark） |
| 迁移 | FluentMigrator |
| 前端 | Vue 3 + Element Plus + Axios |
| 认证 | JWT + PBKDF2-SHA256 + RBAC |

## 项目结构

```
src/
├── AutoTest.Core/          领域模型、接口、枚举（无外部依赖）
├── AutoTest.Application/   应用服务、编排器、Pipeline、DTO
├── AutoTest.Execution/     HTTP/TCP/DB/Python 执行引擎 + DSL 步骤执行器
├── AutoTest.Assertions/    断言评估器（每种目标类型独立实现）
├── AutoTest.Dsl/           DSL 解析、模板变量替换、Schema 校验
├── AutoTest.Orchestration/ DSL 运行时引擎（DAG 执行、断路、进度持久化）
├── AutoTest.AI/            AI 客户端（OpenAI 兼容）
├── AutoTest.Infrastructure/ 仓储（Dapper）、HostedService、Outbox、SignalR Hub
├── AutoTest.Migrations/    FluentMigrator 迁移脚本
├── AutoTest.Webapi/        ASP.NET Core 主机、Controller
├── Auth/                   JWT 签发/验证、RBAC、刷新令牌
├── common/CacheCommons/    防击穿 MemoryCache
├── common/LockCommons/     Redis 分布式锁（自动续期）
AutoTestWeb/                Vue 3 前端
examples/                   DSL 模板示例（9 个）
```

依赖方向：`Webapi → Infrastructure → Application → Core`，`Execution/Assertions/Dsl/Orchestration/AI → Core`

## 监控目标类型

| 类型 | 说明 | 示例 |
|------|------|------|
| HTTP | HTTP 请求 + 断言 | API 健康检查 |
| TCP | TCP 连接 + TLS + 消息收发 | Redis/MySQL 端口探测 |
| DB | SQL 查询/命令/标量 | 订单一致性检查 |
| Python | 脚本进程执行 | 自定义检查逻辑 |
| TEMPLATE | DSL 多步工作流 | 登录 → 查用户 → 校验 |

## 快速开始

```bash
# 1. 配置连接字符串（appsettings.json）
# Database:Provider = SqlServer | Sqlite
# Redis:ConnectionString
# AI:Endpoint + AI:ApiKey

# 2. 运行迁移
dotnet run --project src/AutoTest.Webapi

# 3. 前端
cd AutoTestWeb && npm install && npm run dev
```
